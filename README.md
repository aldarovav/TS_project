# Проект: Влияние нормализации на глобальные модели временных рядов

## Описание
Исследование гипотезы о том, что нормализация рядов важна для глобальных нейросетевых моделей, но влияет неоднозначно на модели разных классов. В работе используются данные M4 (Monthly, 100 рядов), четыре бейзлайна (Naive, SeasonalNaive, auto.theta, auto.ets) и две глобальные модели (CatBoost, PatchTST). Сравнивается качество (sMAPE) без нормализации и с тремя типами скейлинга: StandardScaler, RobustScaler, QuantileTransformer.

## Структура репозитория
- `config.py` - параметры эксперимента
- `run_experiment.py` - главный скрипт для запуска
- `src/` - модули с загрузкой, предобработкой, моделями, оценкой
- `results/` - результаты и ноутбук с анализом
- `requirements.txt` - зависимости
